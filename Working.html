Working

Smart contracts are the protocols or simply the code which is used to enter all the terms of contracts concluded between parties for the parties into blockchain. The participants obligations are given in the smart contract in ‘if-then’ form. Once the given conditions have been met, the smart contracts executes the transactions and ensures the security and integrity of the agreement.[1][a]

Ethereum allows developers to program their own smart contracts, or 'autonomous agents', as the ethereum white paper calls them.

Smart contracts can:

  • Smart contracts can work as multi signature accounts, so that funds are spent only when there is majority
  • It can manage agreement from different users  
  • Provide utility to other contracts
  • It stores information about an application, such as  membership records.
